C和指针-字符串、字符和字节

C
C
Publish Date:   2018-11-04
Update Date:   2018-11-04
Word Count:   786
Read Times:   3 Min
Read Count:  

字符串是一种重要的数据结构，C 语言没有显示的字符串数据类型，字符串以字符串常量或者字符数组形式出现。字符串常量适用于不会对字符串修改的场景。

字符串基础

字符串是一串零个或者多个字符，且以一位模式为 0 的 NUL字节结尾。字符串包含的内部不能出现 NUL 字节，计算长度时也不包括 NUL 字节。

字符串长度

标准库提供了库函数 strlen 的原型：

size_t strlen(char const *string);

但是请注意其返回类型为 size_t 代表一个无符号整数类型。在表达式中使用无符号类型时要注意类型转换和提升。

if(strlen(x)>=strlen(y))... //    A
if(strlen(x)-strlen(y)>=0)...//    B

在 B 中由于无符号类型永远大于0，所以其结果未必符合预期。

不受限制的字符串函数

不受限制指的是这些函数都是通过寻找字符串参数结尾的 NUL 字节判断其长度。

char *strcpy(char *dst,char const *src);
char *strcat(char *dst,char const *src);

有两点值得注意：

  • 如果 dst 和 src 内存重叠，其结果是不可预知的。
  • 程序员必须保证 dst 拥有足够的内存。

长度受限的字符串函数

char *strncpy(char *dst,char const *src,size_t len);
char *strncat(char *dst,char const *src,size_t len);
char *strncmp(char const *s1,char const *s2,size_t len);

注意点：

  • strncpy 不会再 dst 末尾添加 NUL 字节。
  • strncat 会在dst末尾自动添加 NUL 字节。

字符串查找基础

//返回指向字符ch第一次出现的位置的指针
char *strchr(char const *str,int ch);
//返回指向字符ch最后一次出现的位置的指针
char *strnchr(char const *str,int ch);
//返回group字符串任意字符第一次出现的位置指针
char *strpbrk(char const *str,char const *group);
//返回s1 中s2 第一次出现的位置指针
char *strstr(char const *s1,char const *s2);

高级字符串查找

//返回str起始部分匹配group中任意字符的字符数
size_t strspn(char const *str,char const *group);
//返回str起始部分不与group中任意字符匹配的字符数
size_t strcspn(char const *str,char const *group);

内存操作

由于字符串是一个以 NUL 字节结尾，所以如果要操作内部包含NUL的字符串数据，前面的字符串函数将无能为力。不过 C 语言提供了字节层面的操作函数 memxxx

void *memcpy(void *dst,void const *src,size_t length);
void *memmove(void *dst,void const *src,size_t length);
void *memcmp(void const *s,void const *b,size_t length);
void *memchr(void const *s,int ch,size_t length);
void *memset(void const *s,int ch,size_t length);

警告总结

  • 避免在有符号的表达式中使用 strlen 函数。
  • 避免在表达式中混用有符号和无符号数。
  • strcpy 和 strcat 函数要保证源数组内存足够。
